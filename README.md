# Linux Learning Platform

A beginner-friendly Linux learning platform built using HTML, CSS, and GitHub Pages.

## Live Website

[https://chidanandabv.github.io/linux-learning/](https://chidanandabv.github.io/linux-learning-site/)

---

# About The Project

This platform is created to help users learn Linux through:

* Linux tutorials
* Learning notes
* Commands and examples
* Images and diagrams
* Embedded videos
* Beginner to advanced topics

The website is hosted completely free using GitHub Pages.

---

# Features

* Responsive Linux learning website
* Simple UI
* Embedded YouTube videos
* Topic-based learning
* Fast static website
* Free hosting using GitHub Pages

# How To Run Locally

## Clone Repository

```bash
git clone https://github.com/chidanandabv/linux-learning.git
```

## Open Folder

```bash
cd linux-learning
```

## Run Website

Open:

```text
index.html
```

in your browser.
