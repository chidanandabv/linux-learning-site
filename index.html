Linux Basics

Fundamental concepts every Linux administrator must know before diving into commands.

💡 What is Linux? Linux is a free, open-source, Unix-like operating system kernel created by Linus Torvalds in 1991. It powers servers, supercomputers, mobile devices (Android), and embedded systems worldwide.

Linux Filesystem Hierarchy

Understanding the Linux directory structure is fundamental to system administration.

DirectoryPurpose
/Root directory — the top of the filesystem hierarchy
/binEssential user binaries (ls, cp, cat, etc.)
/sbinSystem binaries (fsck, init, iptables)
/etcSystem configuration files
/homeUser home directories
/rootRoot user's home directory
/varVariable data (logs, mail, spool, cache)
/tmpTemporary files (cleared on reboot)
/usrUser programs and data (secondary hierarchy)
/optOptional/add-on software packages
/devDevice files (hardware interfaces)
/procVirtual filesystem for process/kernel info
/sysVirtual filesystem for system hardware info
/bootBoot loader files (kernel, initrd, grub)
/libEssential shared libraries and kernel modules
/mediaMount points for removable media
/mntMount points for temporary filesystems
/srvService data (web, FTP, etc.)
/runRuntime variable data (PIDs, sockets)
💡 Tip: FHS The Filesystem Hierarchy Standard (FHS) defines the directory structure. You can read more with: man hier

Permissions & Ownership

Control who can read, write, and execute files and directories.

📖 Understanding Permissions Linux permissions are divided into three categories: Owner, Group, and Others. Each has Read (r=4), Write (w=2), and Execute (x=1) permissions.

📋 Permission Breakdown

SymbolOctalPermissionFileDirectory
r4ReadView file contentsList directory contents
w2WriteModify fileCreate/delete files inside
x1ExecuteRun as programEnter/traverse directory
chmod
Change file permissions
Beginner

Octal Mode

  • chmod 755 script.shrwxr-xr-x (owner: full, others: read+execute)
  • chmod 644 file.txtrw-r--r-- (owner: read+write, others: read)
  • chmod 700 private_dir/rwx------ (only owner has access)
  • chmod 777 public/rwxrwxrwx (everyone full access) ⚠️
  • chmod +x script.shAdd execute permission
  • chmod -R 755 directory/Recursive permission change

Symbolic Mode

  • chmod u+x fileAdd execute for user (owner)
  • chmod g-w fileRemove write for group
  • chmod o=r fileSet others to read-only
  • chmod a+x fileAdd execute for all
  • chmod u+rwx,g+rx,o+r fileCombined symbolic permissions
chown / chgrp
Change file owner and group
Beginner
  • chown user file.txtChange file owner
  • chown user:group file.txtChange owner and group
  • chown -R user:group dir/Recursive ownership change
  • chgrp group file.txtChange group only
  • chown :group file.txtShorthand to change group only
umask
Set default file permissions
Intermediate
  • umaskShow current umask value (typically 022)
  • umask 022Default: files=644, dirs=755
  • umask 077Restrictive: files=600, dirs=700
  • umask 002Shared: files=664, dirs=775
Special Permissions
SUID, SGID, and Sticky Bit
Advanced
  • chmod 4755 fileSUID — run as file owner
  • chmod 2755 dir/SGID — new files inherit group
  • chmod 1777 /tmpSticky bit — only owner can delete
  • chmod u+s fileSet SUID symbolically
  • chmod g+s dir/Set SGID symbolically
  • chmod +t /tmpSet sticky bit symbolically
ACL (Access Control Lists)
Fine-grained permission control
Advanced
  • getfacl file.txtView ACL entries
  • setfacl -m u:username:rwx fileGrant specific user permissions
  • setfacl -m g:groupname:rx fileGrant specific group permissions
  • setfacl -x u:username fileRemove ACL entry for user
  • setfacl -b fileRemove all ACL entries
  • setfacl -R -m u:user:rwx dir/Recursive ACL setting
  • getfacl file1 | setfacl --set-file=- file2Copy ACL from one file to another

Text Processing & Stream Editors

Powerful tools for searching, filtering, and manipulating text data.

grep / egrep / fgrep
Search text using patterns (Regular Expressions)
Beginner
  • grep "pattern" file.txtSearch for pattern in file
  • grep -i "pattern" file.txtCase-insensitive search
  • grep -r "pattern" /path/Recursive search in directory
  • grep -n "pattern" file.txtShow line numbers
  • grep -v "pattern" file.txtInvert match (exclude pattern)
  • grep -c "pattern" file.txtCount matching lines
  • grep -l "pattern" *.txtList files containing pattern
  • grep -w "word" file.txtMatch whole word only
  • grep -E "pattern1|pattern2" fileExtended regex (OR pattern)
  • grep -A 3 "pattern" fileShow 3 lines after match
  • grep -B 3 "pattern" fileShow 3 lines before match
  • grep -C 3 "pattern" fileShow 3 lines before and after
  • grep -P "\d{3}-\d{4}" filePerl-compatible regex
  • cat file | grep "error" | wc -lChain: grep + count lines
awk
Pattern scanning and processing language
Intermediate
  • awk '{print $1}' file.txtPrint first column
  • awk '{print $1, $3}' file.txtPrint first and third columns
  • awk -F: '{print $1}' /etc/passwdUse colon as delimiter
  • awk '/pattern/ {print $0}' filePrint lines matching pattern
  • awk 'NR==1 {print}' filePrint first line only
  • awk 'END {print NR}' fileCount total lines
  • awk '{sum+=$3} END {print sum}' fileSum values in column 3
  • awk 'NF>0' fileRemove blank lines
  • ps aux | awk '{print $2, $4, $11}'Get PID, %MEM, and command
sed
Stream editor for filtering and transforming text
Intermediate
  • sed 's/old/new/g' file.txtReplace all occurrences (global)
  • sed -i 's/old/new/g' file.txtIn-place replacement
  • sed -i.bak 's/old/new/g' fileIn-place with backup
  • sed '5d' file.txtDelete line 5
  • sed '5,10d' file.txtDelete lines 5 through 10
  • sed '/pattern/d' file.txtDelete lines matching pattern
  • sed -n '5,10p' file.txtPrint lines 5-10 only
  • sed '2i\New Line' file.txtInsert line before line 2
  • sed '2a\New Line' file.txtAppend line after line 2
  • sed 's/.*password.*/REDACTED/' fileRedact sensitive data
cut / sort / uniq / wc
Text manipulation utilities
Beginner
  • cut -d: -f1 /etc/passwdExtract first field with colon delimiter
  • cut -c1-5 file.txtExtract characters 1-5
  • sort file.txtSort lines alphabetically
  • sort -n file.txtSort numerically
  • sort -r file.txtSort in reverse order
  • sort -t: -k3 -n /etc/passwdSort by 3rd field numerically
  • uniq file.txtRemove consecutive duplicates
  • sort file | uniqRemove all duplicates
  • sort file | uniq -cCount occurrences
  • sort file | uniq -dShow only duplicate lines
  • wc file.txtCount lines, words, characters
  • wc -l file.txtCount lines only
  • wc -w file.txtCount words only
tr / tee / xargs
Translate, tee output, and build commands
Intermediate
  • tr 'a-z' 'A-Z' < file.txtConvert to uppercase
  • tr -d '\r' < file.txtRemove carriage returns (Windows → Linux)
  • tr -s ' 'Squeeze multiple spaces into one
  • echo "test" | tee output.txtWrite to file AND stdout
  • echo "test" | tee -a output.txtAppend to file AND stdout
  • find . -name "*.log" | xargs rmBuild and execute command from input
  • cat files.txt | xargs -I {} cp {} /dest/Process each line individually
